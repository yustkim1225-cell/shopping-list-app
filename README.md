# 🛒 Shopping List App

아이템 추가, 삭제, 체크 기능이 있는 간단한 쇼핑 리스트 웹 앱입니다.

## 기능

- ➕ **아이템 추가** — 입력창에 텍스트 입력 후 `+` 버튼 또는 `Enter`
- ✅ **체크(완료)** — 체크박스 클릭으로 완료/미완료 토글
- 🗑 **삭제** — 개별 삭제 및 완료 항목 일괄 삭제
- 🔽 **필터** — 전체 / 미완료 / 완료 필터
- 💾 **로컬 저장** — localStorage에 자동 저장 (새로고침 후에도 유지)

## 실행 방법

`shopping-list.html` 파일을 브라우저에서 열기만 하면 됩니다. 별도 서버 불필요.

## 테스트

```bash
npm install
node shopping-list.test.js
```

Playwright 기반 자동 테스트 20개 포함 (추가/삭제/체크/필터/영속성).
